    Sobre mi

    Mi nombre es Diarte , Matias Ezequiel soy programador web : full stack con una experiencia de 6 años. Trabajo de forma autónoma por medio de plataformas como Workana o Freelancers y para clientes de otro origen. Tengo conocimiento en el área de Back-End por medio de PHP, utilizando patrones de diseño : MVC , Active record , singleton , entre otros. Uso el framework Laravel para facilitar y darle Velocidad/Calidad a los proyectos que los requieren. Así también se utilizar todas aquellas bondades de Laravel como Eloquent , Passport para seguridad y demás. E trabajado creando API RESTFUL tanto con Laravel como con php solo. Estas Api las consumía con Vue o con Angular dándole a la aplicación la capacidad de SPA (Single page application) lo cual mejora mucho la experiencia de usuario.
